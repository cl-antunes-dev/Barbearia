Projeto demonstrativo de site institucional desenvolvido para uma barbearia moderna, com foco na apresentação dos serviços, fortalecimento da identidade visual e facilidade de agendamento.

A página apresenta serviços como cortes masculinos, barba, acabamento, tratamentos capilares e combos, além de informações sobre a barbearia, equipe, horários de atendimento, localização e canais de contato.

O projeto foi construído com HTML5, CSS3 e JavaScript, aplicando conceitos de responsividade, navegação intuitiva, estilização de componentes e integração com WhatsApp para agendamentos. Desenvolvido para fins de estudo, prática e composição de portfólio.
